# Product Overview

BeadsOps is an internal manufacturing operations dashboard for Skyla (天亮醫療器材), a medical device company that produces chemical beads products.

The system monitors and manages the beads production pipeline:
- Titration (滴定) — dispensing chemicals via IVEK pumps and ports
- Freeze-drying (凍乾) — lyophilization of titrated beads
- Dispensing staff assignment (配藥人員派工)
- Work order tracking, scheduling, and warehouse intake (入庫)
- Abnormal event monitoring and alerting
- Beads IPQC (in-process quality control) data
- Production yield statistics

The UI is a dark-themed, data-dense operations dashboard with real-time polling. All user-facing text is in Traditional Chinese (zh-TW). Code comments are also predominantly in Chinese.
